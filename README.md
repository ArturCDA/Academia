# Academia

Este projeto consiste no desenvolvimento de um sistema web para gerenciamento de uma academia, criado com o objetivo de aplicar, na prática, os conceitos de banco de dados, Java Web e arquitetura MVC.

A aplicação permite o cadastro, listagem, edição e remoção de alunos, centralizando as principais informações de forma simples e organizada. Toda a persistência de dados é realizada em um banco PostgreSQL, garantindo integridade e confiabilidade das informações.

O sistema foi desenvolvido utilizando Java com Maven, separando claramente as responsabilidades entre as camadas de modelo, controle e persistência. A comunicação com o banco de dados é feita por meio de uma classe DAO, enquanto os Servlets controlam o fluxo das requisições e as páginas JSP são responsáveis pela interface com o usuário.

Para melhorar a experiência do usuário, foi aplicado um CSS simples e funcional, proporcionando uma interface limpa, organizada e fácil de navegar, sem o uso de frameworks externos.

Este projeto demonstra a implementação completa de um CRUD funcional, integrando banco de dados e aplicação web, servindo como base sólida para sistemas maiores e mais complexos no futuro
